# libmpsse-rs
Rust binding of libmpsse for FTDI's FT2232 familly of USB chips.

## Dependencies
- [libmpsse](https://code.google.com/archive/p/libmpsse/)
- [libftdi](https://www.intra2net.com/en/developer/libftdi/index.php)

### Pre-Install
ArchLinux
```shell
$ yay -S libftdi libmpsse
```
